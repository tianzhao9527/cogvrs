## 🌐 访问方式

### 1. 在线访问
```bash
# 克隆仓库
git clone https://github.com/tianzhao9527/cogvrs.git
cd cogvrs

# 查看项目手册
open index.html  # macOS
# 或双击index.html文件在浏览器中打开
```

### 2. 运行项目
```bash
# 设置环境
python3 -m venv venv
source venv/bin/activate
pip install numpy matplotlib pygame

# 运行模拟器
python3 run_cogvrs.py

# 验证修复
python3 test_fixes.py
```
